# Scopes

This repo uses folders as scope.

A domain idea is not stable until it lives in the correct scope folder.

## Cascade

```text
00-inbox
10-atomic-domains
20-domain-families
30-composite-loops
40-host-bridges
50-render-descriptors
60-incubation-suites
70-promotion-candidates
80-blocked-or-held
```

Move ideas down the cascade only when their scope becomes clearer.
